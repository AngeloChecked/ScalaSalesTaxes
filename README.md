# ScalaSalesTaxes

Scala solution of [sales-taxes-problem](https://github.com/xpeppers/sales-taxes-problem).

## Getting Started

### Prerequisites 


- Intellij IDEA*
- IDEA Scala plugin*
- sbt 

(*) optionals

### Installing

```shell
git clone https://github.com/AngeloChecked/taxweb.git
```

## Running the tests

```shell
sbt test
```
