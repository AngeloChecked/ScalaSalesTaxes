# ScalaSalesTaxes

Scala solution of [sales-taxes-problem](https://github.com/xpeppers/sales-taxes-problem).

## Getting Started

### Prerequisites 

```
- Intellij IDEA*
- IDEA Scala plugin*
- sbt 
- git

(*) optionals
```

### Installing

```shell
git clone https://github.com/AngeloChecked/ScalaSalesTaxes.git
```

## Running the tests

```shell
sbt test
```
